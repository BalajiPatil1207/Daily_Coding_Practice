## ⚡ Unified Start (Recommended)
Jar tula Dashboard ani sagle ekatra suru karayche asel, tar root folder madhe khali dilleli command wapara:
- **Command:**
  ```powershell
  npm start
  ```
- **Nikal:** Dashboard automatic suru hoil ani tu tuzi progress track karu shakshil.

---

## 1. 📊 Progress Dashboard (React + Tailwind)
Ha landing page mazi overall progress dakhvato.
- **Kuthe:** `dashboard/` folder
- **Run kasa karaycha:**
  ```powershell
  cd dashboard
  npm run dev
  ```
- **Nikal:** Browser madhe `http://localhost:5173` open kara.

---

## 2. 🎨 Frontend Practice (HTML/CSS/JS)
Jar tu pure HTML/CSS/JS files banavlya asatil.
- **Kuthe:** `frontend/vanilla-js/` kiwa `frontend/css-layouts/`
- **Run kasa karaycha:**
  - VS Code madhe file open kara.
  - Right-click kara ani **"Open with Live Server"** select kara.
  - (Requirement: VS Code Live Server Extension).

---

## 3. ⚙️ Backend Practice (Node.js & Express)
API ani server-side logic practice karanyasathi.
- **Kuthe:** `backend/node-express/`
- **Run kasa karaycha:**
  ```powershell
  cd backend/node-express
  node server.js
  ```
  *(Jar nodemon install asel tar `nodemon server.js` wapara).*

---

## 4. 🐍 Python Practice
Python scripts ani automation sathi.
- **Kuthe:** `python/` folder
- **Run kasa karaycha:**
  ```powershell
  python filename.py
  ```

---

## 5. 🧩 Algorithms / DSA
LeetCode solutions test karanyasathi.
- **JavaScript sathi:**
  ```powershell
  node algorithms/leetcode/problem_name.js
  ```
- **Python sathi:**
  ```powershell
  python algorithms/leetcode/problem_name.py
  ```
